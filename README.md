---

##  프로젝트 하이라이트

-  **STT 기반 음성 명령 입력**
-  **LLM 기반 의미 파싱 및 RC카 제어 명령 생성**
-  **TTS 피드백 출력**
-  **Three.js 기반 RC카 시뮬레이터 제어**
-  **Nest.js 서버 + WebSocket 실시간 통신**

이 프로젝트는 **AI + IoT + 실시간 제어**라는 복합적인 기술 요소를 완성도 높게 구현하여  
2025 SW중심대학 페스티벌에서 **최우수상**을 수상하였습니다.
