##  프로젝트 하이라이트

-  **STT 기반 음성 명령 입력**
-  **LLM 기반 의미 파싱 및 RC카 제어 명령 생성**
-  **TTS 피드백 출력**
-  **Three.js 기반 RC카 시뮬레이터 제어**
-  **Nest.js 서버 + WebSocket 실시간 통신**

이 프로젝트는 **AI + IoT + 실시간 제어**라는 복합적인 기술 요소를 완성도 높게 구현하여  
2025 SW중심대학 페스티벌에서 **최우수상**을 수상하였습니다.

## Command Protocol (parseCommand)
- 통합 명령 포맷 사용: <maum_X>(key=value) 형태로 모든 기능 제어
- RC카 수동 이동 지원: <maum_0> — 전진·후진·좌회전·우회전
- 방 자동 이동 기능: <maum_1> — 거실·안방·주방 경로 자동 주행
- 자연어 알람/타이머 처리: <maum_2> — “내일 아침 8시”, “10분 뒤” 등 한국어 시간 파싱
- RC카 속도 제어: <maum_3> — 이동속도 증가/감속
- 휴대폰 찾기 기능: <maum_4> — 사운드 또는 Shockwave 효과 재생
- 콘솔 테스트 가능: parseCommand('<maum_0>(direction=1)') 형태로 즉시 실행
